# Google Sans Flex Fonts

This directory contains the Google Sans Flex font family files for local serving.

## Font Files Required:
- GoogleSans-Flex-Regular.woff2
- GoogleSans-Flex-Medium.woff2
- GoogleSans-Flex-Bold.woff2

## Usage:
These fonts are referenced in the CSS files and served locally to avoid external dependencies.
